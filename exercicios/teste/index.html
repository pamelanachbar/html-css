Quebras de linhas

Para utilizar as quebras de linhas, é necessário utilizar a tag <br>,
vè se que aqui tem uma quebra de linha
Quando deseja-se maior espaçamento, NÂO utilizar várias tags <br>!
Lembrando que mesmo que você quebre as linhas no parábrafo sem utilizar a tag <br>, não ocorrerá as quebras. A tag é quem manda.
