Trabalhando com Listas

Listas ordenadas

Com a tag <ol>, order list cria-se uma lista ordenada e em cada item da lista basta utilizar a tag <li>, list item. Ele automaticamente numera as lista e se algum item for adicionado no meio, ele arruma toods os outros itens. Não é necessário fechar a tag <li>

  1. Wake up
  2. Get up
  3. Brush my teeth
  4. Get breakfast
  5. Go to the school
  1. Wake Up
  2. Get Up
  3. Brush my teeth
  4. Get breakfast
  5. Go to the school

Sendo type igual a "a", faz uma lista a,b,c,d..., sendo I, com números romanos e i com números romanos minúsculos

Por padrão, type começa com 1, mas para modificar começando do zero basta adicionar start=número

  1. Vetor[0]=1
  2. Vetor[1]=2
  3. Vetor[3]=2
  4. Vetor[4]=3

Lista não-ordenadas

Para listar listas não ordenadas, utilizamos a tag <ul>, unordered list, onde temos por padrão o type por disc, mas pode utilizar circle e square

  • Pão
  • Queijo
  • Alface
  • Tomate

Listas de Definições

Para utilizar a Listas de Definições, usa-se a tag <dl>, definition list, dentro temos a tag <dt>, definion term, e a tag <dd>, onde coloca a descrição do termo. É como se fosse um dicionário, definindo termos.

HTML
Linguagem de Marcação para a criação do conteúdo de um site.
CSS
Linguagem de marcação para a criação do design de um site
Java Script
Linguagem de Programação para a criação de interatividade de um site.

O Google dá valor as listas de definições quando é procurado por um termo.
