Usando links externos

Link que aponta para outro site, como youtube, github, replit etc. Ao clicar na frase, te manda nesse site. Com a tag <a>, isso é possivel. Dentro do parametro href (referência de hipertexto), onde coloca-se a url do site inteira dentro.
Exemplo:

Você pode visitar o meu site que fala sobre listas

Percebe-se que quando clicamos no link, o meu site saiu e o outro site ficou na mesma aba. Para clicar no link e criar outra aba, basta criar dois parâmetros, target="_blank" e rel="external"

Você pode visitar o meu site que fala sobre código-fonte e citações

rel="external", link do tipo externo, e target="_blank", quero que tenha um alvo que tenha uma janela em branco

Links Internos

Links internos, faz uma relação com uma página que está dentro do site, servidor. Não precisa colocar a url inteira, apenas criar uma nova página da mesma pasta do index e dentro de href, dar o ctrl espaço que aparece a página.

Essa é a primeira págino do meu site. Se você quiser, pode visitar minha segunda página

Como a segunda página é do meu próprio site, não precisamos utillizar targer, ou simplesmente colocar target="_self"
