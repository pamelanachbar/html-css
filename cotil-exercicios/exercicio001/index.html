Estrutura básica da página HTML

Uma página HTML é um conjunto comandos e textos, digitados em formato de texto, que determina para o navegador o modo de apresentar todos os seus itens (textos, figuras, sons) na tela de um computador, via WWW

Para escrever documentos HTML, só é necessário um editor de textos e conhecimentos sobre os códigos que compõem a linguagem. Os códigos, conhecidos como tags1, indicam a função de cada elemento da página na rede

Cada tag possui 2 símbolos: o sinal de "<" (menor), e o sinal de ">" (maior)

O comando pode ser independente (ou solidário, ou solto) não precisando do fechamento, ou pode precisar de encerramento (também chamado de envolvimentos)

Note que o comando que fecha é o mesmo que abre, mas tem uma barra extra

1 - Tags
